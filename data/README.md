### This folder should contain the datasets.

Data can used in the paper are publicly available
* [Breast Cancer dataset](https://archive.ics.uci.edu/ml/datasets/breast+cancer)
* [Big Five dataset](https://www.kaggle.com/datasets/tunguz/big-five-personality-test?resource=download)
* [FIFA20 dataset](https://www.kaggle.com/datasets/stefanoleone992/fifa-21-complete-player-dataset?select=players_20.csv)

The files should be downloaded in this folder and `breast-cancer.data` should be renamed to `breast-cancer.txt`.
